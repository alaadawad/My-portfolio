My Graduation Project

Advanced Driver Drowsiness Detection System

My graduation project focuses on developing an advanced detection system for signs of driver fatigue using the HuskyLens sensor. This innovative system aims to enhance road safety by providing real-time alerts to drivers when signs of drowsiness are detected.

The project involves implementing machine learning algorithms for accurate detection and analyzing various data inputs to ensure reliable performance. It was presented at the Ninevah University exhibition, received positive feedback, and I was awarded with honors for presenting practical applications in improving driver safety.
